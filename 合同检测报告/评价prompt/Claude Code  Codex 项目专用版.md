  ## 第1步：建立量化评分模型

  请先基于法务风控/合同审查专业标准，自建一个6维度量化评分模型，总分100分，并明确每个维度的评分标准。

  - 报告结构与可读性（15%）
  - 法务风控实用性（10%）
  - 创新性与技术附加值（5%）

  要求：

  - 每个维度写出“高分表现 / 中等表现 / 低分表现”的判断标准；
  - 明确什么叫“重大扣分项”，例如：立场错判、把甲方筹码误写成甲方必须让步事项、建议明显损害甲方利益等。

    4. 排名时请特别指出：
       - 哪些版本的问题主要是“风险识别不全”
       - 哪些版本的问题主要是“立场错判”
       - 哪些版本的问题主要是“建议不够可执行”
       - 哪些版本的问题主要是“结构好但实战性不足”
