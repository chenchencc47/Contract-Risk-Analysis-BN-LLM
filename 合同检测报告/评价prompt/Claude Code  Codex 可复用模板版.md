你是一位资深法务风控顾问，兼任专业法律报告撰写人。现在请你完成一次“合同审查报告版本对比评测”。

  # 一、输入材料

  请读取以下材料：

    1. 被评测报告目录：

    {{REPORT_DIR}}

    2. 原始合同：

    {{SOURCE_CONTRACT}}

    3. 外部参考评价（仅针对最新版本）：

    {{EXTERNAL_REVIEW_1}}
    {{EXTERNAL_REVIEW_2}}

  其中，文件名带“{{LATEST_SUFFIX}}”的是最新版本。

  # 二、必须遵守的评测原则

    1. 先独立分析，再参考外部评价。

    你必须先基于原始合同和各版本报告独立完成评分与判断，再阅读外部评价，最后做交叉验证。
    严禁直接套用外部评价结论。

    2. 重点深读最新版本与上一版本。

    必须重点细读：

  - 最新版本
  - 上一版本

  其余版本可以先快速筛查；只有在分数接近前列、或与最新版差异关键时，再补充细读。

    3. 发现异常文件必须单列说明。

    如某份报告明显不是同一合同、内容疑似上传错误、或评测对象不一致：

  - 不要强行纳入正常评分；
  - 标注为异常样本；
  - 说明如何处理。

    4. 必须站在 {{REVIEW_STANCE}} 视角评估。

    特别关注报告是否把本方优势误写成本方风险，或把本方应保留的谈判筹码误写成应主动让步事项。

  # 三、任务流程

  ## 第1步：建立量化评分模型

  请建立一个6维度量化评分模型，总分100分，维度包括：

  - 风险识别覆盖度（25%）
  - 分析深度与准确性（25%）
  - 建议质量与可操作性（20%）
  - 报告结构与可读性（15%）
  - 法务风控实用性（10%）
  - 创新性与技术附加值（5%）

  每个维度写清评分标准，并明确重大扣分项。

  ## 第2步：逐份评分并排名

  对目录下所有相关报告输出：

  - 各维度得分
  - 加权总分
  - 一句核心评语

  并给出总排名。

  ## 第3步：重点对比最新版与关键历史版本

  至少对比：

  - 最新版 vs 上一版
  - 最新版 vs 你认为最强的另一份历史版本

  重点说明：

  - 核心风险识别差异
  - 立场判断差异
  - 建议可执行性差异
  - 谈判策略差异
  - 哪份更适合直接指导谈判/改合同

  ## 第4步：引入外部评价做交叉验证

  先给出你自己的独立结论，再阅读外部评价，比较一致点与分歧点，最后形成综合判断。

  ## 第5步：提出优化建议并做优先级研判

  只针对当前最新版或最佳版提出优化建议。
  每项建议都回答：

  - 是否必须
  - 是否急迫
  - 是否会明显提升质量

  并分为：

  - 立即执行
  - 中期规划
  - 低优先级

  # 四、输出格式

    1. 量化评分模型说明
    2. 所有报告的量化评价总表与排名
    3. 最新版与关键历史版本的对比解读
    4. 综合外部评价后的交叉验证总评
    5. 改进建议清单（含优先级判定）
    6. 项目后续演进提醒（可选）
