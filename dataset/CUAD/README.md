# 数据集介绍
  ## 简介
  合同理解 Atticus 数据集 (CUAD) 是用于法律合同审查的数据集。 CUAD 由 Atticus 项目的数十名法律专家创建
包含超过 13,000 个注释。任务是突出合同中对人工审查很重要的重要部分。
