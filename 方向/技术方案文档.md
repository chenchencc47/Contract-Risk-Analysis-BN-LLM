## 1. 项目概述

### 1.1 项目背景

传统合同审查依赖人工逐条阅读，存在效率低、成本高、易出错等问题。纯LLM方案虽然提高了效率，但存在以下问题：
- 幻觉问题：可能生成不存在的法律条款或风险点
- 不可解释性：难以追溯风险评估的推理过程
- 置信度评估不足：无法量化不确定性

### 1.2 项目目标

构建一个结合贝叶斯网络与LLM的合同风险评估系统，实现：
1. **提高准确性**：相比纯LLM方案，风险评估准确率提升15-25%
2. **增强可解释性**：提供清晰的风险推理路径和概率分析
3. **量化不确定性**：给出风险概率和置信度评估
4. **降低成本**：个人开发者可承受的成本（年成本<¥10,000）

### 1.3 核心创新点

1. **混合架构**：LLM负责语义理解，贝叶斯网络负责概率推理
2. **可解释性**：通过贝叶斯网络的条件概率表展示推理过程
3. **不确定性量化**：输出风险概率分布而非简单分类
4. **低成本实现**：利用开源工具和免费API，个人开发者可独立完成

### 1.4 项目范围

#### 1.4.1 包含功能
- 合同文本预处理与条款分割
- LLM特征提取（条款类型、关键实体、风险指标）
- 贝叶斯网络风险推理
- 风险评估报告生成
- 简单的Web界面（Streamlit）

#### 1.4.2 不包含功能
- 复杂的文档格式解析（PDF、Word等）
- 用户管理与权限控制
- 大规模并发处理
- 企业级部署方案

### 1.5 成功标准

| 指标 | 目标值 | 测量方法 |
|------|--------|----------|
| 风险评估准确率 | ≥80% | 专家标注对比 |

##### CPT 3: P(综合风险评分 | 违约风险, 合规风险, 操作风险)

使用Noisy-OR模型简化：

```python
def calculate_overall_risk(breach_risk, compliance_risk, operation_risk):
    """
    使用Noisy-OR模型计算综合风险
    
    公式: P(综合风险=高) = 1 - (1-p1)(1-p2)(1-p3)
    """
    base_risk = 0.1
    
    risk_weights = {
        '低': 0.1,
        '中': 0.4,
        '高': 0.7
    }
    
    p1 = risk_weights[breach_risk]
    p2 = risk_weights[compliance_risk]
    p3 = risk_weights[operation_risk]
    
    overall_probability = 1 - (1-p1)*(1-p2)*(1-p3) + base_risk
    
    return min(overall_probability, 1.0)
```

#### 3.3.5 推理算法

```python
推理流程:
1. 变量消元法 (Variable Elimination)
   - 适用于小型网络 (15节点)
   - 时间复杂度: O(n * d^w)
   - 空间复杂度: O(d^w)
   
   其中:
   - n: 变量数量
   - d: 最大取值基数
   - w: 树宽度

2. 推理步骤:
   a) 根据证据固定变量
   b) 消除无关变量
   c) 计算目标变量的后验概率
   d) 归一化概率分布
```

class ResultFusion:
    """
    结果融合器
    
    策略:
    1. 一致性检查
    2. 冲突检测
    3. 置信度加权
    4. 最终决策
    """
    
    def fuse_results(
        self,
        llm_result: Dict,
        bn_result: Dict,
        confidence_threshold: float = 0.7
    ) -> Dict:
        """
        融合LLM和贝叶斯网络结果
        
        参数:
            llm_result: LLM风险评估结果
            bn_result: 贝叶斯网络推理结果
            confidence_threshold: 置信度阈值
        
        返回:
            {
                'final_risk_score': 65,
                'risk_level': '中高风险',
                'confidence': 0.85,
                'consistency': True,
                'conflicts': [],
                'reasoning_path': [...]
            }
        """
        pass
    
    def detect_conflicts(
        self,
        llm_risk: str,
        bn_risk: str
    ) -> List[str]:
        """检测冲突"""
        conflicts = []
        
        risk_levels = {'低': 0, '中': 1, '高': 2}
        llm_level = risk_levels.get(llm_risk, 1)
        bn_level = risk_levels.get(bn_risk, 1)
        
        if abs(llm_level - bn_level) >= 2:
            conflicts.append(f"LLM判断为{llm_risk}风险，BN判断为{bn_risk}风险")
        
        return conflicts

    # 计算指标
    accuracy = accuracy_score(ground_truth, predictions)
    precision = precision_score(ground_truth, predictions, average='weighted')
    recall = recall_score(ground_truth, predictions, average='weighted')
    f1 = f1_score(ground_truth, predictions, average='weighted')
    
    print(f"准确率: {accuracy:.2%}")
    print(f"精确率: {precision:.2%}")
    print(f"召回率: {recall:.2%}")

        'hybrid': [],
        'ground_truth': []

        llm_result = pure_llm_analysis(contract)

        results['hybrid'].append(hybrid_result)
        
        results['ground_truth'].append(true_label)
    
    # 对比分析
    llm_accuracy = accuracy_score(results['ground_truth'], results['pure_llm'])
    hybrid_accuracy = accuracy_score(results['ground_truth'], results['hybrid'])
    
    improvement = (hybrid_accuracy - llm_accuracy) / llm_accuracy * 100
    
    print(f"纯LLM准确率: {llm_accuracy:.2%}")
    print(f"混合方案准确率: {hybrid_accuracy:.2%}")
    print(f"提升幅度: {improvement:.2f}%")
