========================================
  BN Contract Risk Analysis — 启动指南
========================================

项目现在支持两种前端模式：

启动环境
conda activate agent

━━━ 方式一：React 全栈前端（推荐，端口 5173）━━━

1. 启动后端 FastAPI（终端 1）：
   PowerShell:
   cd E:\myProgram\BN-Contract-Risk-Analysis
   .\.venv\Scripts\Activate.ps1
   python -m uvicorn backend.main:app --port 9527 --reload

2. 启动前端 React（终端 2）：
   PowerShell:
   cd E:\myProgram\BN-Contract-Risk-Analysis\frontend
   npm run dev

3. 浏览器打开 http://localhost:5173


━━━ 方式二：Streamlit 调试页（端口 8502）━━━

PowerShell:
cd E:\myProgram\BN-Contract-Risk-Analysis
.\.venv\Scripts\Activate.ps1
python -m streamlit run .\src\contract_risk_analysis\demo\streamlit_app.py --server.port 8502


━━━ 方式三：FastAPI 独立（端口 9527，可调 API）━━━

PowerShell:
cd E:\myProgram\BN-Contract-Risk-Analysis
.\.venv\Scripts\Activate.ps1
python -m uvicorn backend.main:app --port 9527 --reload

浏览器打开 http://localhost:9527/docs 查看 API 文档


━━━ 端口占用处理 ━━━

netstat -ano | findstr <端口号>
taskkill /PID <进程号> /F
